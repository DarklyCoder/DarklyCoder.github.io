部分手机开启Log日志方法

发表于 2016-05-09   |   分类于 开发技巧   |     |  

部分手机为了提升性能默认是关闭日志输出的，这样就会给开发调试带来些许不便，那么怎么打开日志输出呢？

华为

  • 进入拨号界面输入：*#*#2846579#*#*
  • 依次选择ProjectMenu->后台设置->LOG设置->LOG开关 点击打开

酷派

  • 进入拨号界面输入：*20121220#
  • 依次选择 日志输出级别 -> Java log level -> LOGD 即可。
